# WanAndroid
WanAndroid 是根据鸿神大神提供的WanAndroid开放Api来制作的个人App，基本实现了所有的功能，采用Java语言，基于Material Design+AndroidX + MVP +  ViewPager + Okhttp + Glide + SmartRefreshLayout + Circleimageview + tiny等优秀的开源框架开发，注释超详细，方便大家练手。
